Переменные в Python

Переменные — это как коробки, в которые можно положить данные, чтобы потом их использовать. У каждой коробки есть имя, и вы можете обращаться к данным, которые в ней лежат, просто называя её.

С помощью переменных можно:

  • Сохранять данные (текст, числа, результаты вычислений).
  • Использовать их многократно в программе.
  • Заменять данные в любой момент.

Правила именования переменных

  • Имя переменной должно начинаться с буквы или символа подчёркивания (_).
  • Имя не может начинаться с цифры.
  • Можно использовать буквы, цифры и символ подчёркивания, например, user_name, age1.
  • Переменные чувствительны к регистру: Name и name — это разные переменные.
  • Не используйте зарезервированные слова (например, if, while, for), так как они уже имеют своё значение в Python.

Типы данных в Python

Данные, которые вы храните в переменных, могут быть разных типов. Вот основные из них:

  • Числа (int, float):
    • int — целые числа (например, 1, -5, 100).
    • float — числа с плавающей точкой (например, 3.14, -2.7).
  • Строки (str):
    • Это текстовые данные. Они заключаются в кавычки, например: "Привет", 'Python'.
  • Списки (list):
    • Это набор данных, которые хранятся вместе. Например: [1, 2, 3] или ["яблоко", "банан", "вишня"].
  • Словари (dict):
    • Это данные в виде "ключ — значение". Например: {"name": "Alice", "age": 25}.

Каждый тип данных используется для разных задач. Например, для текста — строки, для подсчёта — числа. Некоторые операции работают только с определёнными типами данных. Например, вы не можете сложить текст и число без преобразования.
